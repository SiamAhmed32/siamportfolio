# Kaosar Ahmed Portfolio

Job-first personal portfolio built with Next.js, TypeScript, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy to Vercel

1. Push this folder to GitHub.
2. Import the repository in Vercel.
3. Keep default settings.
4. Deploy.

## Where to edit

Most content is inside:

```text
data/site.ts
```

Important folders:

```text
public/profile/kaosar-ahmed.jpeg   -> profile image
public/projects/                   -> project screenshots
public/resume/Kaosar_Ahmed_CV.pdf  -> downloadable CV
app/page.tsx                       -> page sections
app/globals.css                    -> design system and global styles
```

## Current positioning

Primary target: recruiters, hiring managers, technical interviewers.
Secondary target: business owners who may want serious websites or custom software.
